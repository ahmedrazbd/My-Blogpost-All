By Ahmed Raz • 5 Min Read • June 2026 Main Video Tutorial Here: Watch Now
এই টিউটোরিয়ালে, আমরা দেখাবো কীভাবে আপনি প্রতি মাসে ফ্রিতে বিলিয়ন এআই টোকেন ক্লেইম করতে পারেন। সম্পূর্ণ নির্দেশাবলী এবং কমান্ড সহ, আপনি সহজেই এই সুযোগটি কাজে লাগাতে পারবেন। চলুন শুরু করা যাক!

Needed Command

git clone https://github.com/tashfeenahmed/freellmapi.git
cd freellmapi
npm install
cp .env.example .env
ENCRYPTION_KEY="$(node -e 'console.log(require("crypto").randomBytes(32).toString("hex"))')"
printf "ENCRYPTION_KEY=%s\nPORT=3001\n" "$ENCRYPTION_KEY" > .env
npm run dev
